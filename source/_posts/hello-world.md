## Quick Start

### Create a new post

``` bash
$ hexo new "My New Post"
```


### Run server

``` bash
$ hexo server
```

### Clean cache files

```bash
$ hexo clean

```

### Generate static files

``` bash
$ hexo generate
```

### Deploy to remote sites

``` bash
$ hexo deploy
```

### Create new pages

```bash
$ hexo new [layout] <title>

```

More info: [Hexo_docs](https://hexo.io/zh-cn/docs/)
